# [Orion Play](https://orion-play.netlify.app/)

Video library for space related stuff.

# Tech Stack

- React Js
- Backend from [Mockbee](https://mockbee.netlify.app/)

# Features Added

- Home page with category section.
- Video listing page with category filter.
- Playlist Management
- Like/Dislike
- Watch Later
- History
- Authentication
  - Sign-up
  - Login
  - Logout
- Search video
- Single Video Page
- Notes on a video
- Loading & Alerts
